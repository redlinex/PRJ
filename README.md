Импорт проекта
===
```
idea build.gradle
```
Gradle plugin в Intellij Idea должен быть установлен.
File -> Import -> `build.gradle` 

Запуск
===
```
gradlew.bat tasks
gradlew.bat build
gradlew.bat test
gradlew.bat run
```
